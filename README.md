# 🕰️ TimeTravel Agency - Webapp Interactive

Webapp immersive permettant de simuler une agence de voyage temporel fictive.  
L’utilisateur peut explorer différentes époques et interagir avec un assistant IA conversationnel nommé Æon.

---

## 🛠️ Stack Technique

- React (Next.js App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- API Routes (Next.js /api/chat)
- Déploiement : Vercel

---

## ✨ Features

- 🌍 Landing page immersive avec storytelling
- 🧭 3 destinations temporelles interactives :
    - Paris – 1889
    - Crétacé supérieur
    - Florence – 1504
- 💬 Chatbot IA conversationnel (Æon)
- ⚡ Suggestions rapides de destinations temporelles
- 🎞️ Animations fluides avec Framer Motion
- 📱 Interface responsive (mobile & desktop)

---

## 🤖 IA Utilisées (transparence)

- **Génération de code** : GitHub Copilot
- **Chatbot IA (Æon)** : Mistral AI (via API)
- **UX writing / micro-copy** : assistance IA (ChatGPT / itérations de design)
- **Optimisation UI/UX** : aide à la conception via IA générative
